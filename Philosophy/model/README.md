# Philosophy/model 文件夹说明

## 📁 文件夹结构

本文件夹包含Ontology哲学体系的所有结构化模型文档，采用多种思维表征方式，提供系统化的知识组织。

**文档总数**：17个核心模型文档（含 2025-02 新增：概念判断树、公理推理树、技术选型决策树）
