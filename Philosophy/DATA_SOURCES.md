## 七、数据使用指南

### 7.1 数据引用格式

在文档中引用数据时，使用以下格式：

```markdown
[数据内容]（来源：[数据来源]，更新时间：YYYY-MM-DD，可信度：⭐⭐⭐⭐⭐）
```

**示例**：

```markdown
Palantir在2025年Q3实现营收$X.XB（来源：Palantir官方财报，更新时间：2025-11-XX，可信度：⭐⭐⭐⭐⭐）
```

### 7.2 数据可信度说明

- **⭐⭐⭐⭐⭐（极高可信度）**：官方财报、权威学术文献、经典文献
- **⭐⭐⭐⭐（高可信度）**：行业研究报告、知名媒体、官方技术文档
- **⭐⭐⭐（中等可信度）**：企业公开资料、技术博客、第三方分析
- **⭐⭐（低可信度）**：传闻、未验证的第三方信息

### 7.3 数据更新流程

1. **数据收集**：从可靠来源收集最新数据
2. **数据验证**：验证数据的准确性和时效性
3. **文档更新**：更新相关文档中的数据
4. **记录更新**：更新本文档的"最后更新时间"
5. **交叉检查**：确保所有引用该数据的位置都已更新
