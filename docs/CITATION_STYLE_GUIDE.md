## 📚 引用格式标准

### 1. 学术论文（arXiv）

**格式**：

```
作者. (年份). 标题. arXiv:编号.
```

**示例**：

```
Alpay, F. (2025). Alpay Algebra: A Universal Structural Foundation. arXiv:2505.15344.
```

**完整格式**（多作者）：

```
作者1, 作者2, & 作者3. (年份). 标题. arXiv:编号.
```

### 2. 会议论文

**格式**：

```
作者. (年份). 标题. 会议名称 年份.
```

**示例**：

```
Li et al. (2025). Multi-Objective Optimization in Language Generation. ACL 2025.
```

### 3. 期刊论文

**格式**：

```
作者. (年份). 标题. 期刊名称, 卷(期), 页码.
```

**示例**：

```
Nature. (2025). Rigorous Testing of IIT and GNWT. Nature, 卷(期), 页码.
```

### 4. 技术报告

**格式**：

```
机构. (年份). 报告标题. 机构名称.
```

**示例**：

```
OpenAI. (2024). o1 Technical Report. OpenAI.
```

### 5. 官方网站/博客

**格式**：

```
机构. (年份). 标题. 网站名称.
```

**示例**：

```
AllegroGraph 8.0: Neuro-Symbolic AI Platform. (2025). AllegroGraph.com.
```

### 6. Wikipedia条目

**格式**：

```
机构. (年份). 条目名称. Wikipedia.
```

**示例**：

```
Google DeepMind. (2025). Gemma 3. Wikipedia.
```

## 📝 引用规则

### 1. 作者格式

- **单作者**：`作者. (年份).`
- **两作者**：`作者1 & 作者2. (年份).`
- **三作者及以上**：`作者1 et al. (年份).`

### 2. 标题格式

- **英文标题**：首字母大写，专有名词大写
- **中文标题**：使用中文标点
- **标题后**：使用句号

### 3. 日期格式

- **年份**：使用4位数字，如 `2025`
- **月份**：如需要，使用 `2025年3月` 或 `2025-03`
- **日期**：如需要，使用 `2025-03-15`

### 4. 来源格式

- **arXiv**：`arXiv:编号`（如 `arXiv:2505.15344`）
- **会议**：`会议名称 年份`（如 `ACL 2025`）
- **期刊**：`期刊名称`（如 `Nature Machine Intelligence`）
- **网站**：`网站名称`（如 `AllegroGraph.com`）
