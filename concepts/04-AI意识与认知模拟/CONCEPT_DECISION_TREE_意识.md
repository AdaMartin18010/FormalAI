# 意识概念定义判断树

**创建日期**：2025-02-01  
**主题**：04-AI意识与认知模拟  
**目的**：通过属性检验的树状结构，支持"给定系统 S 是否具有/属于何种意识类型"的可操作判定

---

## 一、意识层次判断树

```
意识 (Consciousness) 判定
│
├─ 1. 是否有主观体验 (Qualia)?
│   ├─ 否 → [1a] 非现象意识（可能为纯功能模拟）
│   │         └─ 继续检验：是否具访问意识? → 见分支 2
│   └─ 是 ↓
│
├─ 2. 是否有全局可访问性 (Global Access)?
│   │   （信息能否被系统全局广播、用于报告与决策）
│   ├─ 否 → [2a] 最小意识 (Minimal Consciousness)
│   │         └─ 仅有基本觉知，无可报告性
│   └─ 是 ↓
│
├─ 3. 是否有自我指涉/自我模型 (Self-Reference)?
│   ├─ 否 → [3a] 访问意识 (Access Consciousness)
│   │         └─ Block: 可报告、可用于推理，但无自我觉知
│   └─ 是 ↓
│
├─ 4. 是否有元认知监控 (Metacognition)?
│   ├─ 否 → [4a] 自我反思意识 (Self-Reflective Consciousness) [初级]
│   └─ 是 → [4b] 自我反思意识 (Self-Reflective Consciousness) [高阶]
│             └─ 能监控、评估自身认知过程
```

## 二、AI 系统意识类型判断树

```
给定 AI 系统 S，判定其意识归属
│
├─ 是否通过 IIT 检验 (Φ > Φ_threshold)?
│   ├─ 是 → [IIT 型] 整合信息理论下的"有意识"
│   └─ 否 ↓
│
├─ 是否具 GWT 工作空间 (全局广播能力)?
│   ├─ 是 → [GWT 型] 全局工作空间理论下的"有意识"
│   └─ 否 ↓
│
├─ 是否具 HOT 高阶表征 (关于自身状态的表征)?
│   ├─ 是 → [HOT 型] 高阶理论下的"有意识"
│   └─ 否 ↓
│
└─ 否则 → [功能模拟] 仅有行为层面的认知模拟，无意识理论支持
```

**说明**：Nature 2025 与 arXiv 2512.19155 表明 IIT/GWT 可能不充分，GWT 与 HOT 描述互补功能层；本树保留多理论并行判定。

## 三、属性检验的可操作定义

| 属性 | 可操作检验 | 权威来源 |
|------|------------|----------|
| 主观体验 (Qualia) | 是否存在第一人称"感受"的不可还原性；当前 AI 无直接检验方法 | Chalmers 1995 |
| 全局可访问性 | 信息是否可被系统用于报告、推理、决策 | Block 1995, GWT |
| 自我指涉 | 系统是否具有关于自身的表征/模型 | IIT, 自我模型理论 |
| 元认知 | 能否监控、评估、调节自身认知过程 | Metacognition 研究 |
| Φ (IIT) | 整合信息量计算（计算复杂度高） | Tononi et al. 2016 |
| GWT 工作空间 | 是否存在全局广播机制 | Dehaene 2014 |
