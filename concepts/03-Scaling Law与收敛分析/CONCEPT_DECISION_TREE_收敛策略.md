## 一、收敛层级归属判断树

```
给定领域/产品 D，判定其收敛层级 (L0–L4)
│
├─ 1. 收敛度 C < 0?（采用标准方案的产品比例下降）
│   ├─ 是 → [L0 反向分化] 场景撕裂，多种方案并存且分化加剧
│   └─ 否 ↓
│
├─ 2. 收敛度 0 < C ≤ 0.60?
│   ├─ 是 → [L1 未收敛] 百家争鸣，无主导方案
│   └─ 否 ↓
│
├─ 3. 收敛度 0.60 < C ≤ 0.80?
│   ├─ 是 → [L2 收敛进行时] 双寡头竞争，2–3 方案主导
│   └─ 否 ↓
│
├─ 4. 收敛度 0.80 < C ≤ 0.95?
│   ├─ 是 → [L3 准收敛] 事实垄断，1–2 方案主导
│   └─ 否 → [L4 完全收敛] 工业标准，单一方案主导 (C > 0.95)
```

## 二、Scaling Law 策略选择决策树

```
给定预算与场景，选择 Scaling Law 策略
│
├─ 1. 推理成本是否为主导成本?（高 QPS、长上下文）
│   ├─ 是 → [Sardana 策略] 训练更小模型、更长 token
│   │         └─ 引用 arXiv 2401.00448；固定推理预算下成本-性能比
│   └─ 否 ↓
│
├─ 2. 是否追求训练阶段计算最优?
│   ├─ 是 → [Chinchilla 策略] D_opt ∝ N^0.74，约 20 tokens/param
│   │         └─ 引用 Hoffmann 2022；与 Figure 1 数值对比
│   └─ 否 ↓
│
├─ 3. 是否复现 Kaplan vs Hoffmann 差异?
│   ├─ 是 → [Porian 三因素] 检查 last layer cost、warmup、optimizer
│   │         └─ 引用 NeurIPS 2024；消融实验验证
│   └─ 否 → [Kaplan 原始] L(N) ∝ N^{-α}，单一参数量幂律
│             └─ 引用 Kaplan 2020；拟合 loss-N 曲线
```
